## 8. 커밋 컨벤션

|          | 사용 예시               |
| -------- | ----------------------- |
| feat     | 기능 구현               |
| design   | CSS 스타일링            |
| remove   | 파일 삭제               |
| move     | 디렉토리 또는 파일 이동 |
| chore    | 패키지 설치             |
| refactor | 리팩토링                |
| fix      | 버그 수정               |
